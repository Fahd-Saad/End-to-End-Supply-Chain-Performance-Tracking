# 📊 Olist E-Commerce Data Analysis

## 📌 Project Overview
This project analyzes customer purchasing behavior, delivery performance, and revenue trends to identify business improvement opportunities.

## 🎯 Business Problem
The company is experiencing customer dissatisfaction due to delivery delays and potential revenue leakage across different states.

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Excel
- SQL
- Table

## 📂 Dataset
The dataset contains information about:
- Orders
- Customers
- Payments
- Products
- Delivery time

## 📊 Key Insights
- Late deliveries increase cancellation rates.
- São Paulo generates the highest revenue.
- Repeat customers contribute significantly to total revenue.
- Certain states show high revenue but low customer retention.

## 💡 Recommendations
- Improve delivery time monitoring.
- Focus retention campaigns on high-value states.
- Optimize logistics for underperforming regions.

## 🚀 Conclusion
Enhancing delivery efficiency and customer retention strategies can significantly improve revenue stability and customer satisfaction.
